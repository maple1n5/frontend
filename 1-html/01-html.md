# HTML

HTML 超文本标记语言 —— HyperText Markup Language。

- 超文本：链接
- 标记：标记也叫标签，带尖括号的文本



双标签：成对出现的标签，如 `<h1>标题<h1>`

单标签：只有开始标签，没有结束标签，如 `<hr>`



HTML 骨架

- html 整个网页
- head 网页头部
- title 网页标题
- body 网页主体

~~~html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>

</body>
</html>
~~~
